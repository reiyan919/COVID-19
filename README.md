COVID-19 Data Analysis
 Project Overview
Analysis of global COVID-19 data with interactive Tableau dashboard showing cases, deaths, and trends.

 Dashboard Features
Key Metrics: Total cases, deaths, death rate

Global Trends: Monthly case/death trends

World Map: Geographic distribution

Country Rankings: Top 10 countries

Correlation Analysis: Relationships between metrics

 Tools Used
Python (Pandas, Matplotlib, Seaborn)

Tableau for visualization

Jupyter Notebook
